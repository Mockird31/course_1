◆ compare_time()

int compare_time ( const char * time1,
const char * time2 )

Сравнивает два времени в формате "HH:MM".

Parameters
time1Указатель на строку, содержащую время в формате "HH:MM".
time2Указатель на строку, содержащую время в формате "HH:MM".
Returns
int Результат сравнения времени:
  • Если время1 позже времени2, возвращает FIRST_DATE_LATER.
  • Если время1 раньше времени2, возвращает SECOND_DATE_LATER.
  • Если времена равны, возвращает DATE_EQUALS.
  • Если произошла ошибка при разборе времени, возвращает код ошибки (res).
