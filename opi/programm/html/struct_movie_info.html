Detailed Description

Структура данных для хранения информации о фильме.

Эта структура содержит поля для хранения различных атрибутов фильма, таких как название (title), жанр (genre), режиссер (director), год выпуска (year), страна производства (country), продолжительность (duration) и возрастное ограничение (age_restriction).
