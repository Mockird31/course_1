◆ validateAgeRestriction()

bool validateAgeRestriction ( const char * age_restriction)

Проверяет, соответствует ли указанная возрастная ограничение допустимым значениям.

Функция сравнивает переданное значение возрастной ограничения с заранее определенным списком допустимых значений. Если переданное значение совпадает с одним из допустимых, функция возвращает true, иначе - false.

Parameters
age_restrictionСтрока, содержащая возрастную ограничение для проверки.
Returns
true, если возрастное ограничение допустимо, иначе false.

bool validateDuration ( int duration)

Проверяет, что продолжительность находится в допустимом диапазоне.

Функция проверяет, что переданная продолжительность (в минутах) находится в пределах от 40 до 180 минут включительно.

Parameters
durationПродолжительность в минутах, которую нужно проверить.
Returns
true, если продолжительность находится в диапазоне от 40 до 180 минут включительно; иначе false.
