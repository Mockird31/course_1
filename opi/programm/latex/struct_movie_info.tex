\doxysubsection{Detailed Description}
Структура данных для хранения информации о фильме. 

Эта структура содержит поля для хранения различных атрибутов фильма, таких как название ({\itshape title}), жанр ({\itshape genre}), режиссер ({\itshape director}), год выпуска ({\itshape year}), страна производства ({\itshape country}), продолжительность ({\itshape duration}) и возрастное ограничение ({\itshape age\+\_\+restriction}).
